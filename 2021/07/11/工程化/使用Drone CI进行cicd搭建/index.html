其中 homepage 和 callback url 填写规则为

  • HomePage 是 DroneCI 的访问地址，若是 Drone 由本地部署，那就可以设置为http://127.0.0.1

  • Authorization callback URL 是 DoneCI 的登陆地址，格式必须是{{HomePage}}/login，如http://127.0.0.1/login
